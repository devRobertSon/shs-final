🕘 과거 발행본 복구 도구

실수로 이전 상태를 발행해서 공지·점수 등이 사라졌을 때, 과거 발행본(커밋)의 내용을 열어 보고 관리 페이지에 다시 입력할 수 있게 돕는 읽기 전용 도구입니다. 비밀번호와 복호화된 내용은 이 브라우저 안에서만 처리되며, 밖으로는 GitHub에 파일을 요청하는 것 외에 아무것도 전송하지 않습니다. 이 페이지는 아무것도 저장하거나 발행하지 않습니다.

커밋 찾기: GitHub 저장소 페이지 → 파일 목록 오른쪽 위 커밋 개수(시계 아이콘 History) 클릭 → 되돌아가고 싶은 시각의 "발행: …" 커밋을 열고 주소창의 주소를 그대로 복사해 붙여넣으세요.
